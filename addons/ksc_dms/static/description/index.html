Document Tags Management

Overview

Our Application helps you to properly manage the document's tag in odoo and even while transferring files from one folder to another.

How It Works

In default odoo if you transfer a file from one folder to another the tags are been removed from the file. Our application manages to transfer all the file data when moved from one folder to another.
