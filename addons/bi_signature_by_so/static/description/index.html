Salesperson Signature in Sale Order Odoo App

Digital Signature by Salesperson in SO Odoo App

Salesperson Signature in Sales Order Odoo app enables salespersons to digitally sign sales orders directly within the Odoo platform. This signature serves as verification and confirmation of their involvement in the sales process, adding a layer of authenticity to the transaction. By requiring a salesperson's signature before a sales order is finalized, businesses can ensure that the order has been reviewed and approved by the responsible sales representative.

Features

Add Signature In Sales Order

User can add signature or drow signature into sales order.

Restrict to Save Order Without Signature

User are not allowed to save sales order without adding signature.
