POS Discount Limit

Set discount limit on product and product category to restrict when apply discount on pos



App Features
  • Set discount limit on product
  • Set discount limit on product category
  • Restrict to apply discount more than discount limit
  • Compatible with community and enterprise
POS Discount Limit

You need to activate discount limit feature in the pos config


You can set discount limit on product


You can set discount limit on product category

Now when user apply discount more than discount limit, system will restrict to apply discount (if you have set discount limit on product and category, then system will first check discount limit on product)
